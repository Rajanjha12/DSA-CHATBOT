# DSA AI Bot 🤖

DSA AI Bot is an AI-powered chatbot designed to help users learn and practice Data Structures and Algorithms (DSA). Users can ask questions related to arrays, linked lists, trees, graphs, sorting algorithms, dynamic programming, and more.

## Features

- Interactive chatbot interface
- Ask DSA-related questions
- Explains algorithms and concepts
- Provides coding solutions
- Clean and responsive UI
- Easy integration with AI APIs (Gemini/OpenAI)

## Technologies Used

- HTML
- CSS
- JavaScript
- AI API (Gemini/OpenAI)
